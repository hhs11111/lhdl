# 骑士传说 - 2D刷宝游戏

一款类似《元气骑士前传》的2D像素风格ARPG刷宝手机游戏，使用 **Godot 4.6** 开发。

## 🎮 游戏特色

- **四大职业**：战士、法师、弓箭手、刺客
- **技能系统**：每个职业拥有独特的技能组合
- **刷宝机制**：击败敌人随机掉落装备
- **装备品质**：普通、优秀、稀有、史诗、传说
- **波次战斗**：越来越强的敌人挑战
- **终极技能**：积攒能量释放强力大招

## 🛠️ 技术栈

- **引擎**：Godot 4.6
- **语言**：GDScript
- **平台**：iOS / Android / PC

## 📁 项目结构

```
knight-quest/
├── assets/           # 游戏资源
├── prefabs/          # 预制件
├── scenes/           # 场景文件
├── scripts/          # 游戏脚本
│   ├── Player.gd     # 玩家角色
│   ├── Enemy.gd      # 敌人AI
│   ├── Weapon.gd     # 武器系统
│   ├── Armor.gd      # 护甲系统
│   ├── Skill.gd      # 技能类
│   ├── DropSystem.gd # 掉落系统
│   ├── GameUI.gd     # 游戏UI
│   └── ...
├── project.godot     # 项目配置
└── README.md
```

## 🎯 操作说明

### PC 端
- **WASD**：移动
- **空格**：攻击
- **1/2/3**：技能
- **4**：终极技能
- **ESC**：暂停

### 移动端
- **虚拟摇杆**：移动
- **攻击按钮**：攻击
- **技能按钮**：释放技能

## 🚀 快速开始

1. 安装 **Godot 4.6**
2. 克隆项目：`git clone https://github.com/hhs11111/lhdl.git`
3. 打开 `project.godot`
4. 点击运行
