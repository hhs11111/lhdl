# 骑士传说 - 2D刷宝游戏

一款类似《元气骑士前传》的 2D 像素风 ARPG 刷宝游戏，使用 **Godot 4.6** 开发。

## 游戏特色

- **四大职业**：战士、法师、弓箭手、刺客，每个职业独特的技能和玩法
- **波次战斗**：一波接一波的敌人挑战，难度递增
- **刷宝掉落**：击败敌人随机掉落金币、经验、药水、武器、宝石
- **装备系统**：5 种稀有度的武器（普通→传说）
- **终极技能**：积攒能量释放强力大招

## 操作说明

### PC 端

| 按键 | 功能 |
|------|------|
| **W/A/S/D** | 移动 |
| **空格** 或 **鼠标左键** | 普通攻击 |
| **1** / **2** / **3** | 技能 1 / 技能 2 / 技能 3 |
| **4** | 终极技能 |

### 移动端

虚拟摇杆 + 触摸按钮（需在 Godot 中进一步定制）

## 运行方式

### 1. 安装 Godot

下载并安装 **Godot 4.6**（或更高版本）：
- 官网：https://godotengine.org/

### 2. 打开项目

- 启动 Godot → 点击 **Import**（导入）
- 选择 `project.godot` 文件
- 点击 **Run**（▶ 播放）按钮开始游戏

### 3. 打包发布

在 Godot 中选择 `Project -> Export`，选择平台（PC / Android / iOS），一键打包。

```
├── README.md              # 说明文档

│   ├── Player.gd          # 玩家角色控制
│   ├── Enemy.gd           # 敌人 AI + 掉落
│   ├── Gold.gd            # 金币道具
│   ├── ExpOrb.gd          # 经验球道具
│   ├── HealthPotion.gd    # 血药道具
│   ├── Gem.gd             # 宝石道具
│   ├── WeaponPickup.gd    # 武器掉落
│   ├── GameScene.gd       # 游戏主场景逻辑
│   └── MainMenu.gd        # 主菜单
└── scenes/                # 场景文件 (.tscn)
    ├── MainMenu.tscn      # 主菜单
    ├── GameScene.tscn     # 游戏场景
    ├── Player.tscn        # 玩家
    ├── Enemy.tscn         # 敌人
    ├── Gold.tscn          # 金币
    ├── ExpOrb.tscn        # 经验球
    ├── HealthPotion.tscn  # 血药
    ├── Gem.tscn           # 宝石
    └── WeaponPickup.tscn  # 武器
```
